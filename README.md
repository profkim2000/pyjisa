# pyjisa
6시간동안 지사에서 완전 초보에게 파이썬 교육하기

## 개요

- 교육시간: 1일 6시간

- 주제: 지사에서 파이썬 활용

- 대상: 파이썬을 전혀 모르는 왕초보 지사 직원

- 목표: 지사에서 개발, 파이썬을 전혀 모르는 직원들에게 파이선 사용법과 지사에서의 파이썬 활용법 제안

<br />

- 파이썬 개요

- 파이썬 설치

- 파이썬 맛보기 - 파이썬을 자비스처럼

- 지사에서의 파이썬 활용

### 2교시

- 프로그래밍이란?
  - 숫자
  - 이벤트
  - 화면 구성

- 프로그래밍을 하면 좋은 점
  - 사업기회가 늘어남 - 인사처에서 만든 프로그램

- 요즘은 AI가 코딩을 다 해 준다는데
  - 어떻게든 만들어 주기는 하는데 내가 원한 게 아닐 수도 있고 수정이 어렵다.
  - 기본적인 코딩을 할 줄 아는 상태에서 AI가 만들어준 코드를 가져다 쓰면 정말 강력해짐. 
  - 그래서 코딩을 배워야 함.
  - 코딩 일을 시키려면 코딩 기본은 알아야.
  - 측량 경력이 없는 사람이 지사장이 되면 안되는 이유와 같음

- 프로그래밍의 종류
  - 데이터베이스
  - 게임
  - GIS

- 프로그래밍 언어
  - C언어
  - 자바
  - 파이썬
