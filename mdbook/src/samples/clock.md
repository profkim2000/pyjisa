### 🕒 실시간 시계 코드

```python
import time
from datetime import datetime

try:
    while True:
        # 현재 시간 가져오기 (년-월-일 시:분:초 형식)
        now = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
        
        # \r (캐리지 리턴)을 사용하여 커서를 줄 맨 앞으로 이동하고 시간을 출력합니다.
        # end=""를 주어 줄바꿈이 일어나지 않도록 합니다.
        print(f"\r현재 시간: {now}", end="", flush=True)
        
        # 1초 동안 대기
        time.sleep(1)

except KeyboardInterrupt:
    # 사용자가 Ctrl + C를 누르면 안전하게 종료합니다.
    print("\n👋 시계를 종료합니다.")

```

---

### 💡 코드 포인트 설명

* **`datetime.now().strftime(...)`**: 현재 날짜와 시각을 우리가 보기 편한 형태의 문자열로 변환합니다. `Y`는 년, `m`은 월, `d`는 일, `H`는 시(24시 형태), `M`은 분, `S`는 초를 뜻합니다.
* **`\r`과 `end=""**`: `\r`은 커서를 현재 줄의 맨 처음으로 되돌리는 제어 문자입니다. 여기에 `end=""`(줄바꿈 방지)를 조합하면, 매초마다 새로운 줄을 만들지 않고 **기존에 출력된 시간 위에 새로운 시간을 덮어쓰게** 되어 디지털시계처럼 작동합니다.
* **`flush=True`**: 출력 버퍼를 강제로 비워 실시간으로 화면에 글자가 나타나도록 보장합니다.
* **`try-except KeyboardInterrupt`**: 무한 루프가 돌다가 사용자가 터미널에서 `Ctrl + C`를 누르면 깔끔하게 작별 인사를 건네며 종료됩니다.
